宏定义说明

#define MB_TCP_PORT_USE_DEFAULT   0

Modbus TCP 默认使用端口(502)

枚举类型说明

enum eMB_MSMode

Modbus 主从模式 (Master/Slave).

Modbus 协议栈有两种实现,Maseter或Slave. Master发送请求命令,接收应答命令. 从站接收请求命令,发送应答命令.

枚举值
MB_MASTER 

主站模式.

MB_SLAVE 

从栈模式.

enum eMBErrorCode

所有协议栈函数的返回值使用 Errorcodes.

枚举值
MB_ENOERR 

没有错误.

MB_ENOREG 

非法的寄存器地址.

MB_EINVAL 

非法参数.

MB_ENORES 

insufficient resources.

MB_EIO 

I/O 错误.

MB_EILLSTATE 

协议栈错误的状态.

enum eMBExceptionCode

Slave 错误码 eMBExceptionCode .

枚举值
ILLEGAL_FUNCTION 

非法的功能码.

ILLEGAL_DATA_ADDRESS 

非法的数据地址.

ILLEGAL_DATA_VALUE 

非法的数据值.

SLAVE_DEVICE_FAILURE 

从设备故障.

ACKNOWLEDGE 

确认.

SLAVE_DEVICE_BUSY 

从栈设备繁忙.

MEMORY_PARITY_ERROR 

奇偶校验错误.

GATEWAY_PATH_UNAVAILABLE 

网关路径不可用.

GATEWAY_TARGET_DEVICE_FAILED_TO_RESPOND 

目标设备无法响应.

enum eMBMode

Modbus 串行传输模式 (RTU/ASCII).

Modbus 支持两种传输模式, ASCII 或 RTU. RTU传输速度较快,但是对硬件更多的要求同时要求网络 低噪音. ASCII传输速度较慢但是更可靠.

枚举值
MB_RTU 

RTU 传输模式.

MB_ASCII 

ASCII 传输模式.

MB_TCP 

TCP 传输模式.

enum eMBRegisterMode

对寄存器的读写操作.

枚举值
MB_REG_READ 

读寄存器的值.

MB_REG_WRITE 

修改寄存器的值.
